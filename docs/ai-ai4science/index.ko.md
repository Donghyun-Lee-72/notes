# AI와 AI4Science

이 섹션은 AI의 핵심 개념을 근거 검색, 도구 사용, workflow 실행, 과학 연구 지원 시스템으로 연결합니다.

## 기초

언어 모델, 토큰화, Transformer, 컨텍스트 윈도, 임베딩, 프롬프팅부터 시작합니다. 이후 시스템이 사용하는 표현과 추론 계층을 설명합니다.

## 시스템 구축

RAG, 재순위화, 구조화 출력, 도구 사용, Agent memory, workflow, AI Agent를 다룹니다. 권한 부여, 검증, 실행, 종료 조건은 모델이 아니라 애플리케이션 코드가 소유해야 한다는 원칙을 따릅니다.

## AI4Science

AI4Science는 이러한 능력을 과학적 발견, 시뮬레이션, 실험 계획, 문헌 종합, 데이터 분석에 적용합니다. 유창한 출력만으로는 과학적 타당성을 보장할 수 없습니다. 추적 가능한 근거, 단위와 조건, 불확실성, 재현 가능한 도구, 과학적 정답에 대한 평가가 필요합니다. 향후 재료과학, 화학, 과학 장비와의 연결을 확장합니다.

## 평가

근거성, LLM 평가, RAG 평가 문서를 사용해 출처 품질, 검색, 추론, 생성, 도구 실행 실패를 구분합니다.
