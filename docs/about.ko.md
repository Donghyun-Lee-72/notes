# AI Notes 소개

AI Notes는 언어 모델, AI Agent, 검색 증강 생성, 평가를 설명하는 공개 지식 모음입니다. Donghyun Lee가 교육 자료이자 기술 학습 기록으로 관리합니다.

이 사이트는 개념을 실제 시스템 설계와 연결하는 간결한 설명을 지향합니다. 원 논문, 공식 문서, 모델별 안전 지침을 대체하지 않으며, 중요한 주장은 독자가 확인할 수 있도록 공개 출처를 제시합니다.

## 편집 원칙

- 용어뿐 아니라 작동 원리를 설명합니다.
- 모델 동작과 애플리케이션·인프라 동작을 구분합니다.
- 중요한 한계와 실패 유형을 명시합니다.
- 공개된 1차 자료와 공식 문서를 우선합니다.
- 평가와 증거를 시스템 설계의 일부로 다룹니다.
- 도구나 근거가 바뀌면 문서를 수정합니다.

## 범위

현재 범위는 LLM 기초, Agent와 RAG 패턴, 주요 구현 도구, 평가입니다. 개인 기록, 비공개 프로젝트, 원본 자료 보관소, 운영 정보는 포함하지 않습니다. 영어판을 원본으로 유지하고 한국어판은 의미와 기술 용어를 대조 검수합니다.
