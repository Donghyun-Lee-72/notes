# About Notes

Notes is a public collection with two main sections: AI and Materials Science &
Chemistry. AI4Science sits within AI and connects its methods to scientific
questions. The collection is maintained by Donghyun Lee as an educational
resource and an evolving record of technical study.

The site favors compact explanations and direct links between related ideas. It
is not intended to replace primary papers, official documentation, or
model-specific safety guidance. References are included when they make a claim
easier to verify or a topic easier to explore in depth.

## Editorial Principles

- Explain the mechanism, not only the terminology.
- Link to an existing explanation instead of repeating it.
- Separate model behavior from application and infrastructure behavior.
- State important limitations and failure modes.
- Use public primary sources and official documentation when a citation is useful.
- Treat evaluation and evidence as part of system design.
- Revise pages when the underlying tools or evidence change.

## Scope

The current scope covers LLM fundamentals, agent and RAG patterns, evaluation,
AI4Science, materials characterization, and chemistry foundations. Personal notes, private project records,
raw source archives, and operational material are outside the scope of this
site.
