# 재료과학

재료과학은 **공정–구조–물성–성능**의 연결을 다룹니다. 이 섹션은 원자 결합과 결정구조에서 시작해 미세구조, 열역학, 속도론, 분석법, 공학적 거동으로 지식을 확장합니다.

## 분석법

- [전자현미경: SEM과 TEM](../learn/electron-microscopy-sem-tem.md) — 전자–물질 상호작용, 산란과 신호 생성, 영상, 회절, 분광, 시료 준비, artifact, 장비 선택을 설명합니다.

## 확장 예정 지도

- 원자구조, 결합, 결함, 확산
- 상평형, 상태도, 재료 열역학
- 기계적·전자적·광학적·자기적·열적 물성
- X선 회절, 역공간, 구조인자, 쌍상관
- 표면, 계면, 박막, 나노구조

이 섹션은 공개적으로 독립 검증 가능한 참고문헌을 사용합니다. 비공개 교재는 주제 선정과 페이지 단위 검증에는 활용하지만, 원문이나 PDF를 공개·재배포하지 않습니다.
