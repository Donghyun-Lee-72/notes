# AI4Science

AI4Science는 AI 모델과 AI 시스템을 과학적 발견, 시뮬레이션, 실험 계획, 문헌 종합, 데이터 분석에 적용합니다. 과학 응용도 이 가이드에서 설명하는 모델 기초, 검색 시스템, 도구, workflow와 평가 방법에 의존하므로 AI 섹션 안의 전문 영역으로 구성합니다.

## 과학적 요구사항

유창한 출력만으로 과학적 타당성을 보장할 수 없습니다. AI4Science 시스템에는 추적 가능한 출처, 명시적인 단위와 조건, 불확실성, 재현 가능한 도구, 과학적 관측이나 합의된 정답에 대한 평가가 필요합니다.

## 도메인 지식과의 연결

향후 문서는 이러한 시스템 요구사항을 재료과학, 화학, 과학 장비, 시뮬레이션, 자동화 실험과 연결합니다. 해당 도메인의 개념은 [재료과학·화학](../materials-science/index.md) 섹션에서 구성합니다.
