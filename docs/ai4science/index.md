# AI4Science

AI4Science applies AI models and engineered AI systems to scientific discovery,
simulation, experiment planning, literature synthesis, and data analysis. It is
a focused area within the AI section because scientific applications depend on
the same foundations, retrieval systems, tools, workflows, and evaluation
methods described elsewhere in this guide.

## Scientific requirements

Fluent output is not sufficient evidence of scientific validity. AI4Science
systems need traceable sources, explicit units and conditions, uncertainty,
reproducible tools, and evaluation against scientific observations or accepted
ground truth.
